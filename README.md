# FileCompressor

A simple file compression and decompression tool written in C++.

## Requirements

- C++20 or later

## Usage

### Encode

```bash
./file_compressor.exe -e <input_file_path> <output_file_path>
```

### Decode

```bash
./file_compressor.exe -d <input_file_path> <output_file_path>
```
